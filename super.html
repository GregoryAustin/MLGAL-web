Evolving Convolutional Neural Networks to Identify Galaxies in the Zone of Avoidance

David Jones

Abstract

The Zone of Avoidance makes it difficult for astronomers to catalogue galaxies at low latitudes to our galactic plane due to high star densities and extinction. However, having a complete sky map of galaxies is important in a number of fields of research in astronomy. There are many unclassified sources of light in the Zone of Avoidance and it is therefore important that there exists an accurate automated system to identify and classify galaxies in this region. This study aims to evaluate the efficiency and accuracy of using an evolutionary algorithm to evolve the topology and configuration of convolutional neural networks to automatically identify galaxies in the Zone of Avoidance. A supervised learning method is used with data containing near-infrared images. The input image resolution and number of near-infrared passbands needed by the evolutionary algorithm is also analyzed while the accuracy of the best evolved Convolutional Neural Network (CNN) is compared to other attempts at creating an automated program for the same purpose.

Project Objectives

The program Galyxi Vysion, which uses an EA to modify CNN topologies, was developed to identify galaxies in the ZoA. The objectives for this study are to determine if Galyxi Vysion can:

▪ Accurately identify galaxies in the ZoA.

▪ Deal with “noise” from other light sources in the ZoA.

▪ Produce a CNN configuration that is highly specific to the task of identifying galaxies at given input dimensions, making it worth the processing time over a human adjusted configuration.

▪ Potentially offer an improved work-flow to astronomers in helping identifying galaxies in the ZoA.

Datasets

Before testing was done using data from the ZoA, small scale tests were done using the popular CIFAR10 dataset to see if everything was running as it should and that the CNNs both learned, tested, and evolved correctly.

The VISTA and UKIDSS surveys were used for all training and testing data. Objects were labelled as galaxies or non-galaxies. The data was presented in multiple sets of three FITS (Flexible Image Transport System) files, each representing a different passband; J, H, and K.

The data amounted to 34GB. A compressed ZIP of all data used can be downloaded from this external link.

Methods

▪ More input images were created using transformations.

▪ A bounding-box algorithm was used to create subimages focused on all labelled objects.

▪ An evolutionary algorithm was used to modify the parameters of a population of CNNs.

▪ The CNNs were trained and tested on the input subimages. A CNN's fitness is its percentage accuracy when tested.

▪ Different sizes of input subimages and combinations of passbands were tested before evaluating the output data produced.

Results and Conclusions

▪ The best results achieved 91.75% accuracy.

▪ These results were successful in comparison to other attempts to automatically identify galaxies.

▪ Using smaller input image sizes had little effect on the results down to as low as 16x16. However, the best results were still achieved using 32x32 images.

▪ The best results were obtained using all the passbands (JHK).

▪ Using only K images obtained the best results using an individual passband.

▪ Evolutionary convergence took roughly 15 generations using a population size of 100.

▪ The best CNN configuration could not be improved upon by a human configuration.

▪ The approach was shown to be a possible tool for astronomers and could evolve a CNN that could identify potential galaxies in unseen data.
